# Installation Guide

## Prerequisites

- Python 3.11+
- GitHub CLI (`gh`)
- Git

## Installation Methods

### Via pip (recommended)

```bash
pip install refactoring-agent
```

### From source

```bash
git clone https://github.com/yourusername/agent-claude-refactoring.git
cd agent-claude-refactoring
pip install -e ".[dev]"
```

## Verification

```bash
refactoring-agent --version
```
